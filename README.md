# Trabzon Web Sitesi

Bu proje, Trabzon’un tarihi, kültürel ve turistik yerlerini tanıtan bir web sitesidir.

## İçerik

- Anasayfa
- İlçeler
- Tarihi Yerler
- Kültür
- İletişim

## Özellikler

- Video ve görseller ile tanıtım
- Responsive tasarım
- Git LFS ile büyük video dosyaları yönetimi
